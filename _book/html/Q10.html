input 里的 name

input 里的 name

由于表单提交的数据一般都是以key:value的方式提交，所以input中的name属性主要用来标识提交的数据的key值，好让服务器处理程序分辨。 例如：

<input name=name value=adam>
<input name=age value=18>

则服务器在后台收到的数据格式可能是这样的：

{
  "name":"adam",
  "age":18
}
