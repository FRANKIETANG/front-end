内容与样式分离

内容与样式分离

含义：对于内容、结构与表现相分离，最早是在软件开发架构理论中提出来的，XHTML 的标签只用来定义文档的结构，所有涉及表现的东西通通剥离出来，把它放到一个单独的文件里，这个单独的文件就是 CSS 。

作用是：

  1. 数据的多样显示。通过不同的样式表适应不同的设备，做到内容与设备无关。
  2. 保持整个站点的视觉一致性变得非常简单，修改样式表就可以轻松改版；
  3. 由于结构清晰，数据的集成、更新和处理更加方便灵活；
  4. 更有意义的搜索。
